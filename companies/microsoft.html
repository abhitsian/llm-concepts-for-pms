Microsoft on AI

Building the AI platform layer across cloud (Azure AI), productivity (M365 Copilot), and developer tools (GitHub Copilot) -- positioning Azure as the default infrastructure for the largest AI workloads.

What the CEO Said

"More than any transformation before it, this generation of AI is radically changing every layer of the tech stack, and we are changing with it."

Satya Nadella, CEO, FY2025

"Our planet-scale cloud and AI factory, together with Copilots across high value domains, is driving broad diffusion and real-world impact."

Satya Nadella, CEO, FY2025

"The largest AI workloads run on Azure. And when that happens, you learn the workload faster. You optimize the entire platform faster."

Satya Nadella, CEO, Q4 FY2025

"You can think of agents as the new apps."

Satya Nadella, CEO, Q2 FY2026

"I think many investors are doing a very direct correlation between the capex spend and seeing an Azure revenue number... A large share of AI capacity is being allocated to our own products first, such as Microsoft 365 Copilot and GitHub Copilot, before being made available to external Azure customers."

Amy Hood, CFO, Q2 FY2026

The Numbers

  • Azure annual revenue: $75B+ for FY2025, up 34%; Q4 FY2025 Azure growth was 39% YoY
  • AI services contributed 16 percentage points to Azure's growth in Q3 FY2025
  • Azure processed 500+ trillion tokens via Foundry APIs -- up 7x YoY
  • Microsoft Cloud total revenue: $168B annually, up 23%
  • GitHub Copilot: 20M all-time users, 4.7M paid subscribers (up 75% YoY); 90% of Fortune 100 use it
  • Microsoft 365 Copilot: 15M paid seats (up 160% YoY); daily active users up 10x YoY
  • Consumer Copilot daily users growing nearly 3x YoY; 100M-150M monthly active users
  • Total company revenue: $281.7B for FY2025, up 15%
  • 2GW+ of data center capacity stood up in the past year
  • Demand for AI services outstrips data center supply -- all new equipment booked to capacity

AI Products & Initiatives

  • Azure AI / Azure OpenAI Service -- hosting GPT-4, GPT-4o and future models
  • Microsoft 365 Copilot -- Word, Excel, PowerPoint, Teams, Outlook integration
  • GitHub Copilot -- code completion, Copilot Enterprise, Copilot Pro+
  • Copilot consumer -- Windows integration, search, browsing, shopping
  • Azure Foundry -- API platform for model deployment
  • Cosmos DB optimized for AI chat/agent workloads
  • Custom silicon and data center buildout at planetary scale

The AI Strategy

Microsoft is building the most complete enterprise AI stack: infrastructure (Azure), frontier models (OpenAI partnership), productivity (M365 Copilot), and developer tools (GitHub Copilot). The insight that “a large share of AI capacity is allocated to our own products first” reveals the real strategy: Microsoft is its own best customer, using AI capacity to improve its own products before selling it externally.

The platform economics are staggering: 500+ trillion tokens processed (7x YoY), 15M paid M365 Copilot seats (160% YoY growth), 20M GitHub Copilot users with 90% Fortune 100 penetration. Nadella’s “agents are the new apps” positions Microsoft as the platform where agents are built and deployed — just as Windows was the platform where apps ran.

The “every layer of the tech stack is changing” framing justifies the massive CapEx (2GW+ of data center capacity in one year). Microsoft is betting that enterprise AI is a platform shift comparable to cloud, and the company that owns the platform captures the majority of value — regardless of which specific models win.

The Counterintuitive Insight

Microsoft’s biggest AI advantage isn’t the OpenAI partnership — it’s the installed base. 400M+ Office users will get AI through automatic updates, not by choosing a product. GitHub Copilot’s 90% Fortune 100 penetration proves that enterprise distribution, not model superiority, determines the winner. In a world where every model is “good enough,” the company that ships AI through software people already use wins by default.
